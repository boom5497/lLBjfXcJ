# 项目简介
## 核心功能
- 前后端代码及SQL文件
- 万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包
## 适用场景
- 老年人健康数据远程监控
- 健康数据管理
- 健康报告生成
- 健康知识提供
- 系统管理
- 个人中心
## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
